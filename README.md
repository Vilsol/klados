# Klados

## Features

* Multi-cluster management with automatic or manual kubeconfig definition.
* Supports any GVR and CRD with viewing, editing and management.
* Live log tailing with infinite scrollback on-demand with common log format highlighting.
* An xterm based terminal that can be executed into any container.
* YAML editor with schema validation, diff-viewing and indent guides.
* Window split-out for logs, terminal or editor.
* Smart search based on name, labels and annotations. Search saving.
* Plugin support via OCI for GVR visualization, feature extension, permissions, etc.
* Prometheus and metrics-server graph visualizations.
* Port-forwarding with resilient auto-reconnection.
* Templated resource creation.
* Command palette with shortcuts.
* Many QoL knick-knacks to ease the tool use.
* Theming for dark and light mode (for all you eye-burners out there).
* Cluster-wide and per-resource event streams.
* Read-only mode for safe browsing
* Bulk operations on listed resources.
